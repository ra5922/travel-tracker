🌍TRAVEL TRACKER
A full-stack web application that enables users to visually track and document their travel history on an interactive world map. The platform allows users to mark visited countries, attach personal notes, store travel dates, and explore locations via Google Maps integration.

OVERVIEW
Travel Tracker provides an intuitive and engaging way to document travel experiences. With a dynamic SVG-based world map and multi-user support, the application transforms simple travel logging into a visually rich and interactive experience.

KEY FEATURES
Interactive World Map – Clickable SVG-based global map
Dynamic Country Highlighting – Visited countries are color-coded per user
Travel Notes – Store personalized notes for each country
Date Tracking – Record when each country was visited
Hover Tooltips – View country name, notes, and visit date instantly
Google Maps Integration – Navigate to any country directly
Multi-User Support – Manage travel data for multiple users
